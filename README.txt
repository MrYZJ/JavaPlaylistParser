Welcome to Java Playlist Parser  <https://sourceforge.net/projects/jplaylistparser/>
=================================================

Java Playlist Parser is a toolkit for detecting and extracting metadata and
structured text content from various documents using existing parser
libraries.

Getting Started
===============

Java Playlist Parser is based on Java 6 and uses the Apache Ant <http://ant.apache.org/>
build system. To build Java Playlist sParser, use the following command in this directory:

    ant jar

Issue Tracker
=============

If you encounter errors in Java Playlist Parser or want to suggest an improvement or
a new feature, please visit the Java Playlist Parser issue tracker at
https://sourceforge.net/p/jplaylistparser/tickets/. There you can also find the
